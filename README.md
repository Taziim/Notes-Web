# Note Management System

A simple **Note Management System** built with **HTML, CSS, and JavaScript**.  
This application allows users to **add** and **delete** notes easily. All changes are updated in real-time on the page.

## Features

- Add new notes
- Delete existing notes
- Responsive and lightweight
- Easy to use

## Demo

Check out the live demo here: https://taziim.github.io/Notes-Web/

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Open the application in your browser.
2. Enter a note in the input field.
3. Click the **Add** button to add the note.
4. Click the **Delete** button next to a note to remove it.
